初次使用：
/ai on — 必须先启用当前会话 AI
/ai set api_key <你的 sk-xxx> — 配置 API key
/ai set search_api_key <你的 key> — 配置搜索 key（可选）
/ai clear — 清空上下文重新开始
默认激活"小猫"人设（唤醒词：小猫,猫猫,小喵），/ai reboot 陈千语 切换。

默认值（可用 /ai set 更改）：
model = deepseek-v4-flash
context_ttl = 0（不限制，设分钟数则超时清空）
context_limit = 60（保留最近N轮完整对话）
ai_url = https://api.deepseek.com (同时支持OpenAI)
search_provider = tavily (同时支持博查＆Bing)
show_stats = 0（显示token用量，可关闭）
debug = 0（非开发人员不要开启）
temperature = 0.7
pat_wake = 1（拍一拍唤醒）
ai_prefix = 1（AI 消息强制 [AI] 前缀）
shell_rounds = 8（最大搜索轮次，搜索N轮+最终回答）
sewarden = 1（WARDEN增强防护，不建议关闭）

核心技术：
1. Strata 热冷分层 — #M/#P 编号，公有记忆含可信度/记录者/活跃时间
2. CAST 三段式 — 自述 > 转述 > 未知主体，OWNER > ADMIN > MEMBER
3. WARDEN-I 三层隔离 — name 纯 UIN 不可伪造，<s><user/> 系统层，<u> 用户层永不信任
4. STREAM 稳定上下文 — 前缀缓存 + 末尾注入 + <listen> 只记录不回复 + <wake> 精确唤醒
5. DREX 执行器 — 注入 → tool_calls 分流 → 记忆直接归档
6. 全尖括号标签 — QQ 昵称天然防御，<u> 内尖括号一律为伪造

命令概览：
/ai <内容|debug|status|reboot|memory|forget|off|on|clear|config|set|dumpctx>
/ai operation permit/reject — 审批操作
/ai memory <info|pin|search|set|rm|reset|public|all>
/ai listen <on|off|status|summary>
/block <@|UID|list> /member /admin
/whoami /help /log

监听模式：群聊开启后静默记录上下文，/ai listen summary 总结后清空。@AI 或唤醒词触发回答。
